# RAG Pipeline — Retrieval-Augmented Generation

Builds a complete RAG system that loads a PDF, chunks it, generates embeddings, stores them in a FAISS vector store, and answers questions using a HuggingFace LLM.

**Key concepts:** Document loading, text chunking, vector embeddings, similarity search, LLM querying

**Tools:** LangChain, FAISS, HuggingFace Transformers, SentenceTransformers, PyPDF
