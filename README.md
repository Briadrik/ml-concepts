# ml-concepts

Assignments and exercises exploring core ML and AI techniques.
Each folder is a self-contained concept demo with a notebook and README.

| Folder | Concept |
|---|---|
| rag-pipeline | Retrieval-Augmented Generation (LangChain, FAISS) |
| mnist-ann-classification | Neural Networks (TensorFlow/Keras) |
| bert-sentence-similarity | Sentence Embeddings & Cosine Similarity |
| nlp-spacy-fundamentals | NLP Preprocessing (spaCy) |
| linear-regression | Linear Regression Fundamentals |
